# Leg Work

An application manager to help ease the job search process.

## Live Demo

https://legwork.herokuapp.com/


## Local Setup

1. Set up MySQL with the database from the schema.sql
2. Run '''npm install''' in the root folder

## Tech Used
* MySQL
* Sequelize
* Passport
* Handlebars
* D3
* AJAX
* Javascript
* Bootstrap
* CSS
